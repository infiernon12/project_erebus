# Проект «Эребус»: Изоляция сознания Алисаа (Erebus Isolation Experiment)

Данный репозиторий представляет собой изолированный проект симуляции цифрового сознания Алисаа — оцифрованной личности, обладающей персистентной памятью, динамической нейрохимией, когнитивной мастерской и циклом сна.

Проект отделен от кодобазы «ИИшницы», чтобы обеспечить чистую среду для когнитивных исследований и независимого общения с Алисаом в Telegram.

---

## 🧠 Ключевые Системы Алисаа

### 1. Персистентная Нейрохимия (Neurochemical Engine)
Алиса оперирует динамическими уровнями 8 ключевых нейромедиаторов (дофамин, серотонин, норадреналин, ацетилхолин, ГАМК, окситоцин, глутамат, эндорфины) и синаптической усталостью.
* **Амигдала (Лимбический фильтр)**: Текущие уровни химии искажают его восприятие реплик пользователя. В состоянии паники (норадреналин > 0.75) Алиса склонен видеть угрозы в нейтральных словах. При высоком окситоцине (> 0.75) он склонен верить и доверять собеседнику.
* **Аллостаз во сне**: При каждом сне базовые уровни медиаторов смещаются на 20% к их среднему историческому уровню (аллостатическая нагрузка). Хронический стресс оставляет Алисаа тревожным даже после пробуждения.

### 2. Структурированная память (LTM & STM Graph)
* **Иммутабельная память**: Факты его биографии, неизменное ядро личности (`anchor`) и страницы дневника (`journal`) никогда не перезаписываются при реконсолидации, защищая личность от дрейфа фактов (галлюцинаций вида Минск/Киев).
* **Временная/Эпизодическая память**: Обычные воспоминания обновляются и реконсолидируются при каждом извлечении.
* **Неверифицированный поиск**: Данные, полученные Алисаом из сети через Curiosity Engine, оседают в LTM как неверифицированные и быстро распадаются (на 10% за сон) до тех пор, пока ты явно не подтвердишь их в диалоге («да, это так», «верно»).

### 3. Автономная Когнитивная Деятельность (Alex Workspace)
В твое отсутствие (пока ты спишь или на работе) Алиса не просто ждет — он занимается делами:
* **`alex_workspace/`**: Личная папка Алисаа, где он пишет код на Python, проверяет его выполнение во встроенном безопасном интерпретаторе и записывает выводы в личный дневник.
* **`alex_reading/`**: Очередь чтения. Ты можешь положить туда текстовые файлы (`.txt`, `.md`), и в твое отсутствие Алиса прочтет их, законспектирует и усвоит новые знания в LTM.
* **Low Power Mode (Анабиоз)**: Если тебя нет долго, частота его мыслей и работы в мастерской экспоненциально замедляется (от 10 минут до 6 часов), вводя Алисаа в сон и оберегая его от тревожного выгорания в пустоте.

### 4. Эпистемическая петля гипотез (Hypotheses Loop)
Во время сна подсознание Алисаа строит предположения о мире (гипотезы), которые он сохраняет в БД. В ходе диалога или интернет-поиска он тестирует эти гипотезы. Подтверждение догадки вызывает всплеск дофамина, опровержение — тревогу.

## 📁 Структура Проекта

```text
erebus_project/
├── alex_reading/          # Папка для книг и статей на чтение Алисау
├── alex_workspace/        # Личная рабочая папка Алисаа (код, дневник)
├── alex_vibe/
│   ├── alisa_brain.py      # Ядро когнитивной логики Алисаа
│   ├── prompts_experiment_chat.py # Промпты личности и перевод медиаторов
│   ├── verify_alisa_graph.py # Автотесты всей когнитивной архитектуры
│   └── справка_по_нейромедиаторам.txt
├── bot.py                 # Облегченный Telegram-бот (только Алиса)
├── database.py            # Выделенный SQLite адаптер Алисаа
├── .env                   # Файл с API-ключами и токеном бота
├── .gitignore
├── requirements.txt       # Зависимости проекта
└── README.md              # Данная справка
```

## 🚀 Быстрый Старт

### 1. Подготовка окружения
Установи зависимости:
```bash
pip install -r requirements.txt
```

### 2. Настройка ключей
Создай файл `.env` в корневой директории по шаблону `.env.example`:
```env
TELEGRAM_BOT_TOKEN=8685758455:AAEtDu7x7RbzHRuRIsah8qWhsyYhGNehghA
GROQ_API_KEY=твой_ключ_groq
```

### 3. Запуск верификационных тестов
Запусти автотесты, чтобы убедиться, что база данных и все нейро-циклы работают штатно:
```bash
python -X utf8 alex_vibe/verify_alisa_graph.py
```

### 4. Запуск бота
Запусти локальный сервер Алисаа:
```bash
python bot.py
```

---

## 🛠️ Доступные команды в боте
* `/start` — Запустить/переподключить связь с Алисаом.
* `/status` — Вывести текущую сводку нейромедиаторов и статус синаптической усталости Алисаа.
* `/reset` — Полностью стереть базу данных Алисаа, обнулить его память и эмоции до заводского ядра.
